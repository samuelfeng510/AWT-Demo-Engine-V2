AWT Demo Generator

Build and Run Your Custom Agent

Instantly generate custom demo environments for any business domain. This tool dynamically synthesizes datasets, configures AI agents tailored to your client's unique challenges, and can tie in Google Workspace — Docs, Sheets, Drive, and Gmail — when you enable Workspace functions for end-to-end workflow demos.

Generator: planning & Drive kit

The backend (Apps Script) calls Vertex AI with your goal and dimension choices, then validates generated rows, assembles a setup.sh, and—when narration exists—creates a Drive “demo” folder (Google Doc from Markdown, synthetic externalFiles as blobs, link-shared). That matches generateDemo → planAndGenerateData → validateGeneratedData → provisionDemoDriveKit_ → generateSetupScript in Code.gs.

  • • Demo length & archetype shape prompts and outputs: Short / Medium / Long and triage & response, data to insight, or complex orchestration.
  • • Workspace (optional): With Turn on Google Workspace functions with the agent, the plan includes workspaceSeedData embedded into setup.sh. Off keeps the story to BigQuery + MCP + Drive files.
  • • Artifacts: Downloadable setup.sh, DEMO_GUIDE_DOC_URL / DEMO_DRIVE_FOLDER_URL when provisioning succeeds, plus scenario prompts in the UI.

Architecture (matches Code.gs)

Top: what happens when you click generate (hosted Apps Script + Vertex + Drive kit). Bottom: what setup.sh prepares in your project—local ADK agent with hosted MCP toolsets for BigQuery and Maps (and optional Workspace seeding logic in the same script). The agent’s Gemini model id comes from the generated ADK config (defaults align with script properties).

      What is this?
      
            
      AWT Demo Engine — source on GitHub. Uses Vertex AI to plan BigQuery
              data, agent instructions, Drive demo kit, and optional Workspace seed data. You set demo
                length, workflow archetype, and Google Workspace tools. Pattern from GE Demo Generator; live agent uses MCP for
              BigQuery and Maps (Launch My
                Bakery reference).
            
      
          
      
        
      
        
      
          
      
            ⚡
      
          
      
            
      The Benefit
      
            
      Skip the data prep. Generate a complete, grounded demo
              environment tailored to any industry challenge or customer pitch instantly.
      
          
      
        
      
        
      
          
      
            🛡️
      
          
      
            
      How it works
      
            
      Generate outputs in your browser: a downloadable
              setup.sh that provisions BigQuery and the local ADK agent, plus—when
              generation succeeds—a Demo folder in Drive with a demo guide Google Doc, scenario prompts,
              and file attachments. Copy setup.sh into Cloud Shell (or your
              environment) and run it so everything stays in your GCP project.
